Table Headers

Use the TH element to define table headers.

Name Age
Jill Smith 50
Eve Jackson 94

Vertical Table Headers

Set the first table cell in each table row to a TH element for the first column to become table heads

First Name Jill Eve
Last Name Doe Jackson
Age 30 56

Cells that span two rows

To make a cell span more than one row, use the rowspan attribute.
